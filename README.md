# Smart Queue Management System

## 📌 Overview
An IoT-based Smart Queue Management System developed using ESP8266 and Firebase.

## 🚀 Features
- Real-time Queue Monitoring
- Firebase Cloud Integration
- Wi-Fi Communication
- Embedded C Programming

## 🛠 Hardware Used
- ESP8266 (NodeMCU)
- Sensors
- Wi-Fi

## 💻 Software Used
- Arduino IDE
- Firebase
